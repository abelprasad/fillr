# Icons Directory

This directory should contain the extension icons in PNG format:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Generating Icons

### Option 1: Use the Icon Generator
1. Open `icon-generator.html` in your browser
2. Right-click each icon and save as PNG
3. Save with the correct filenames in this directory

### Option 2: Create Custom Icons
Use design tools like:
- Figma
- Canva
- Adobe Illustrator
- Inkscape (free)

Design guidelines:
- Use a clear, simple design
- Primary color: #2563eb (blue)
- Ensure icons are recognizable at small sizes
- Save as PNG with transparent background

### Option 3: Use Online Tools
- Use an SVG to PNG converter
- Use icon generator websites like:
  - flaticon.com
  - icons8.com
  - favicon.io

## Temporary Placeholder

- Create 16x16, 48x48, and 128x128 PNG files
